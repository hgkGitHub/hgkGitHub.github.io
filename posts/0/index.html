git的命令

扩展：

①删除文件

命令：git rm -r –cached 文件名

​ ①git首次安装必须设置一下用户签名，否则无法提交代码

②初始化本地库

命令： git init

③查看本地库状态

命令：git status

④添加到暂存区

命令：git add 文件名

从暂存区中删除，但是本地库中仍有此文件

命令：git rm –cached 文件名

⑤提交到本地库

命令： git commit -m “日志信息” 文件名

查看版本命令

1、命令：git reflog

查看版本的详细信息

2、命令：git log

⑥修改文件

命令：vim 文件名

将修改文件再次加入到暂存区

命令： git add 文件名

查看状态

命令：git status

再次把文件提交到本地库

命令： git commit -m “日志信息” 文件名

⑦版本穿梭

命令：git reset –hard 版本号

二、分支的操作

①创建分支

命令：git branch 分支名

②切换分支

git checkout 分支名

③查看分支

git branch -v

④合并分支

命令：git merge 分支名

合并分支异常

三、远程库

①创建别名

命令：git remote add 别名 远程库名

②查看远程库别名

命令：git remote -v

③删除远程库别名

命令：git remote rm 别名

④推送本地库代码至远程库

命令：git push 远程库名(别名) 分支名

⑤拉取远程库代码到本地库

命令：git pull 远程库名（别名）分支名

⑥克隆远程库代码到本地库

命令：git clone 远程库名
